# Data Structures with Python

This repository contains Python implementations of fundamental data structures.
The project is designed to demonstrate a clear understanding of core data
structures and their underlying algorithms without relying on built-in or
high-level library implementations.

All data structures are implemented from scratch with a focus on:
- Algorithmic correctness
- Code readability
- Conceptual clarity
- Modular structure

This repository serves as a foundational study and practice project for
computer engineering students and entry-level software developers who want
to strengthen their understanding of data structures using Python.

## Project Scope

The repository includes implementations of the following data structures:

- Linked List
- Stack
- Queue
- Trees (Binary Search Tree, AVL Tree, Heap, General Trees)
- Hash Table
- Graph

Each data structure is organized into its own directory to ensure a clean
and maintainable project structure.

## Design Principles

- No external data structure libraries are used.
- Each structure is implemented according to its theoretical definition.
- Code is written to be easy to read and understand rather than overly optimized.
- Recursive techniques are separated into a dedicated module where appropriate.

## Purpose

The main goals of this project are:
- To build a strong foundation in data structures
- To practice implementing algorithms from first principles
- To improve problem-solving and algorithmic thinking skills
- To maintain a clean and professional GitHub repository structure

This repository is intended for educational and self-improvement purposes.

# Python ile Veri Yapıları

Bu depo, temel veri yapılarının Python dili kullanılarak gerçekleştirilen
uygulamalarını içermektedir. Proje, hazır veya üst seviye kütüphanelere
dayanmadan veri yapılarının ve bu yapıların arkasındaki algoritmaların
mantığını açık bir şekilde ortaya koymayı amaçlamaktadır.

Tüm veri yapıları sıfırdan uygulanmış olup şu prensipler esas alınmıştır:
- Algoritmik doğruluk
- Kod okunabilirliği
- Kavramsal açıklık
- Modüler yapı

Bu depo, veri yapıları konusundaki bilgisini güçlendirmek isteyen
bilgisayar mühendisliği öğrencileri ve yazılım geliştirmeye yeni başlayanlar
için bir çalışma ve referans kaynağı olarak hazırlanmıştır.

## Kapsam

Depoda aşağıdaki veri yapılarının Python uygulamaları bulunmaktadır:

- Bağlı Liste (Linked List)
- Yığın (Stack)
- Kuyruk (Queue)
- Ağaçlar (İkili Arama Ağacı, AVL Ağacı, Heap, Genel Ağaçlar)
- Hash Tablosu
- Graf Yapıları

Her veri yapısı, düzenli ve sürdürülebilir bir proje yapısı sağlamak amacıyla
ayrı klasörler altında organize edilmiştir.

## Tasarım İlkeleri

- Harici veri yapısı kütüphaneleri kullanılmamıştır.
- Her yapı teorik tanımına uygun şekilde uygulanmıştır.
- Kodlar aşırı optimizasyondan kaçınılarak anlaşılır olacak şekilde yazılmıştır.
- Rekürsif yapılar, gerektiği durumlarda ayrı bir modül altında toplanmıştır.

## Amaç

Bu projenin temel amaçları şunlardır:
- Veri yapıları konusunda sağlam bir temel oluşturmak
- Algoritmaları temel prensiplerinden yola çıkarak uygulamak
- Problem çözme ve algoritmik düşünme becerilerini geliştirmek
- Düzenli ve profesyonel bir GitHub depo yapısı oluşturmaktır

Bu depo eğitim ve kişisel gelişim amaçlı hazırlanmıştır.
