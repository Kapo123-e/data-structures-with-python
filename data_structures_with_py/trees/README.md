This directory contains implementations of tree-based data structures.

Trees are hierarchical data structures commonly used for efficient searching,
sorting, and data organization.

The following structures are implemented in this directory:
- Binary Search Tree
- AVL Tree
- Heap (Max Heap)
- General Tree structures
- Tree traversal algorithms

Each implementation focuses on clarity and correctness, providing a practical
representation of the theoretical concepts behind tree data structures.

Bu klasör, ağaç tabanlı veri yapılarının Python ile gerçekleştirilen
uygulamalarını içermektedir.

Ağaçlar, hiyerarşik veri yapıları olup arama, sıralama ve veri organizasyonu
gibi işlemlerde yaygın olarak kullanılır.

Bu klasörde aşağıdaki yapılar bulunmaktadır:
- İkili Arama Ağacı (Binary Search Tree)
- AVL Ağacı
- Heap (Max Heap)
- Genel Ağaç Yapıları
- Ağaç dolaşma (traversal) algoritmaları

Her uygulama, ağaç veri yapılarının arkasındaki teorik kavramları
pratik bir şekilde gösterecek biçimde, anlaşılır ve doğru olacak şekilde
hazırlanmıştır.
